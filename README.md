# ⏳ Pomodoro Minimalista

Um cronômetro Pomodoro moderno, minimalista e automatizado, desenvolvido com foco em alta produtividade.

acesse atráves do Link: 

https://btechsolution.github.io/pomodorotimer/

## ✨ Funcionalidades

- **Ciclo Automático**: Alterna entre foco e pausa sem intervenção manual.
- **Ajuste Dinâmico**: Permite configurar os tempos de estudo e descanso em tempo real.
- **Alertas Sonoros**: Emite sinal sonoro de 5 segundos ao final de cada etapa.
- **Interface Responsiva**: Design escuro (Dark Mode) com efeitos de desfoque e transparência.
- **Título Dinâmico**: O tempo restante é exibido diretamente na aba do navegador.

## 🛠️ Tecnologias

- **HTML5**
- **CSS3** (Variáveis e Backdrop-filter)
- **JavaScript** (Manipulação de DOM e Web Audio API)

## 🚀 Como usar

1. Clone este repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Ajuste os minutos de Foco e Pausa conforme sua necessidade.
4. Clique em **Iniciar** e mantenha o foco!
